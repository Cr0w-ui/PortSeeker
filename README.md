Port Scanner, opensource and programmed in C++ for linux distros.

### Free Open-Source vulnerability scanner
#### For arch based distros
```
pacman -S curl
g++ compile.cpp -o prtker -std=c++11 -lcurl
```
#### For debian based distros
```
sudo bash start.sh
./prtker

```
### execute
```
./prtker
```
